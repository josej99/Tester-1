<p class="lead">This page discusses HTML website templates (basically a prebuilt website where you can fill in the blanks).</p>

<a href="https://www.quackit.com/html/templates/download/bootstrap/business-1/index.html" target="_blank"><img class="screenshot md-right" src="/pix/html/templates/download/bootstrap/business-1.png" alt="Business 1 screenshot"></a>

<p>One question many people ask before learning HTML is "Do I have to be good at design?".</p>
<p>The answer to that is "Not at all!".</p>
<p>OK, if you want a career building websites, it can help if you are a good designer. But, this is not essential. Many web developers either outsource their design to a professional designer, or purchase HTML website templates. They prefer to concentrate on <i>building</i> the actual website rather than <i>designing</i> the website.</p>

<p>An HTML website template is basically a prebuilt website, where you, the developer, can modify as required. All the HTML/CSS code and images are included in the template, and a good template will be compatible with most modern browsers.</p>



<h2>Benefits of Using a Website Template</h2>
<p>Here are some benefits of using an HTML website template:</p>
<ul>
<li><strong>Speeds up development time</strong>: You don't need to spend hours designing your website, then converting that into code, then checking for browser compatibility etc.</li>
<li><strong>More professional looking website</strong>: Unless you're a very good designer, it will be difficult to achieve a great looking website every time. And, if you're not a good designer, your websites will reflect this.</li>
<li><strong>Learning</strong>: Using an HTML website template is a great way to learn from other developers. Once you've downloaded the template, you have full access to the code, so you can learn the tricks the developers used in achieving an effect.</li>
<li><strong>Cheap</strong>: Well, cheaper than hiring a professional designer.</li>
</ul>
